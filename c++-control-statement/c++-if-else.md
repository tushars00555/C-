# C++ if-else

In C++ programming, if statement is used to test the condition. There are various types of if statements in C++.

* if statement
* if-else statement
* nested if statement
* if-else-if ladder

## C++ IF Statement

The C++ if statement tests the condition. It is executed if condition is true.

}
